Instructions for Install

If you have a Meta Quest 2 VR headset, you can try GrayBoxVR yourself! Click the link to the left, download the zip file, unzip it and run the game. Make sure you have the Oculus app for your PC, the headset linked up, and the "Unknown Sources" setting in the General tab of the settings menu is enabled. Happy building!
